6. En Packet Tracer, solo el dispositivo Server-PT puede funcionar como servidor. Las PC de escritorio o portátiles no pueden funcionar como servidores. Según lo que estudió hasta ahora, explique el modelo cliente-servidor.

El modelo comprende dos partes el Cliente que es la parte que interactua o solicita la informacion y el Servidor que es el que provee esta informacion. De igual forma cuando se habla de estos no se tiene que referir especificamente a un dispositivo como tal, pero mas bien al rol que representan, dado que en definitiva son ambos capaces de realizar la funciona del otro si son configurados de forma correcta.

9. Explique la diferencia entre una LAN y una WAN, y dé ejemplos de cada una.

Las redes LAN se utilizan para establecer acceso a usuarios dentro de una red geografica pequena como puede ser una pequena oficina o un departamento.
Las redes WAN son caracter mucho mas amplio con una extension de varios kilometros estas no necesariamente tiene que ser particular pero pueden serlo como son las redes internas de una empresa muy grande con varias sucursales o a nivel mas publico que seria una ciudad o un distrito.
